## 🚀 安装与使用

### 📋 系统要求
- **前端**：Node.js 16+，npm 8+
- **后端**：Java 17+，Maven 3.6+
- **数据库**：MySQL 8.0+

### 🚀 快速开始
```bash
# 1. 克隆项目
git clone https://github.com/Jvxi/Forum.git
cd Forum

# 2. 启动后端
cd project-backend
mvn spring-boot:run

# 3. 启动前端
cd project-frontend
npm install
npm run dev
```
